# 🚀 Space Explorer

A beautiful React app that fetches real astronomy images from NASA's public API and displays them in a clean, responsive UI.

## 🌌 Live Preview

> Add your deployed link here

## ✨ Features

- Fetches 12 random images from NASA Astronomy Picture of the Day API
- Beautiful dark space-themed UI
- Loading skeleton cards while data is fetching
- Error state with retry button
- Click any card to open full image with description
- Search/filter by title or description
- Fully responsive on all screen sizes
- Animated star field background

## 🛠️ Tech Stack

- React 18
- Vite
- NASA APOD Public API

## 🚀 Getting Started

### 1. Clone the repo
git clone https://github.com/yourname/space-explorer.git

### 2. Go into the folder
cd space-explorer

### 3. Install dependencies
npm install

### 4. Start the app
npm run dev

### 5. Open in browser
http://localhost:5173

## 📡 API Used

NASA Astronomy Picture of the Day (APOD)
https://api.nasa.gov/planetary/apod?api_key=DEMO_KEY&count=12

No API key required — uses NASA's free DEMO_KEY
